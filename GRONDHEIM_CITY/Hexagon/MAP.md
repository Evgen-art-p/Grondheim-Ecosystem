# ХРАМ — Карта проекта (v2)

## Быстрый старт

1. Скачай папку `temple/` целиком
2. `js/config.js` → замени `YOUR_API_KEY`
3. Положи видео (temple.mp4, appear.mp4, loop.mp4)
4. Открой `index.html` через Live Server
5. Кабинет: `cabinet/index.html`

## Путь человека

```
  «ищи»  →  тьма  →  храм  →  Безликий (appear→loop)
                                    ↓
                        ┌───────────┼───────────┐
                   ключевое     «не знаю»    не распознано
                      ↓             ↓             ↓
                  ХРАНИТЕЛЬ      КОСТЁР       (игнор)
                                    ↓
                            ключевое слово
                           (запоминается!)
                                    ↓
                          Костёр продолжает...
                          «Ты не один» + [ТЫ НЕ ОДИН]
                                    ↓
                              ХРАНИТЕЛЬ
                          (appear→loop + диалог)
                                    ↓
                          «спасибо» или [ТЫ НЕ ОДИН]
                                    ↓
                              ВЫПУСК
                          (release→release_loop)
                                    ↓
                            «ты здесь»
                                    ↓
                          (8 сек тишины)
                                    ↓
                            «ищи» ← снова
                          человек решает сам
```

## Память Храма (memory.js) — НОВЫЙ МОДУЛЬ

Два слоя хранения:

| Слой | Что | Как долго |
|------|-----|-----------|
| **localStorage** | Был ли, сколько раз, какие хранители, темы Костра | Навсегда (даже после закрытия) |
| **sessionStorage** | Текущий хранитель, история Костра этой сессии | До закрытия вкладки |

### Что помнит Безликий
При повторном входе показывает: **«Ты уже был здесь. Что теперь?»**
(как в документе «внутреннее устройство»)

### Что помнит Костёр
Если человек вернулся — в промпт Костра добавляется контекст:
- О каких темах говорили раньше
- Краткая история прошлого разговора (до 10 реплик)

Костёр не повторяет, но может мягко упомянуть: «Мы уже касались страха...»

### Что сохраняется
- `temple_visited` — был ли раньше
- `temple_visits` — счётчик визитов
- `temple_guardian_log` — к кому ходил [{id, date}]
- `temple_fire_topics` — темы Костра ["страх", "лия"]
- `temple_fire_history` — краткая история (sessionStorage)

### Очистка (для тестов)
В консоли браузера:
```js
import('./js/memory.js').then(m => m.memory.clear())
```
Или просто: `localStorage.clear()`

## Маркер [ТЫ НЕ ОДИН]

| Кто | Когда | Результат |
|-----|-------|-----------|
| Костёр | Ключевое слово + прощание | → Хранитель |
| Костёр | Прощание без ключа | → Выход |
| Хранитель | Сам решил отпустить | → Release → Выход |
| Человек | «спасибо/пора/ухожу» | → Хранитель прощается → Release |


---

## Ключевые слова → Хранители

| Слова | Хранитель |
|-------|-----------|
| одиночество, одиноко, одинок, одна, никому | **Лия** |
| вина, виноват, справедливость, несправедливо | **Юст** |
| страх, страшно, боюсь, потеря, потерять, деньги | **Кей** |
| ложь, вру, обманываю, притворяюсь, фальшь | **Виктор** |
| пустота, пусто, бессмысленно, смысл, потерялся | **Финч** |

## Что делает каждый JS-модуль

| Модуль | Роль | Ключевой метод |
|--------|------|----------------|
| config.js | Все настройки | — |
| state.js | State machine | `transition(state, data)` |
| main.js | Дирижёр | `switch(state)` |
| video.js | Видео | `playSequence(intro, loop)` |
| fire.js | Костёр | `send(text)`, `_handleRelease()` |
| guardian.js | Хранитель | `enter(id)`, `send(text)` |
| keywords.js | Маршрутизация | `detect(text)` → guardianId |
| threshold.js | Порог | `start()`, `reset()`, `onTrigger` |
| memory.js | Память | `recordVisit()`, `getFireContext()` |
| cursor.js | Курсор | `initCursor()` |
